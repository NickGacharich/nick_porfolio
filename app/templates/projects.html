Anime Character Generator

In this project I created a DCGAN model to generate fake images from real images. In this model I used animation characters as input in order to create fake animated characters. The project was made in Jupyter Notebook with the torch, numpy, and matplotlib library for creating the model and ploting the performance of the model.
